
# BullsEye
Python/OpenCV pipeline tracks a red ball in video via contour detection, computing area, radius, and timing (video + visible time). Outputs an annotated video with overlay (contour, circle, metrics) and exports all measurements to a CSV file for further analysis and plotting.
